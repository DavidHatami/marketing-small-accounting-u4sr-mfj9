# Atherton and Co - Marketing Website

A marketing website for Atherton and Co, a small accounting firm. The site showcases the firm's services, introduces the team through bios, and provides a consultation form for potential clients to get in touch. Built with Astro and styled with Tailwind CSS for a clean, professional appearance.

## Local Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

Requires Node.js >= 20.

## Deploy to Netlify

1. Connect your repository to Netlify
2. Configure build settings:
   - **Build command:** `npm run build`
   - **Publish directory:** `dist`
3. Deploy

Alternatively, use the Netlify CLI:

```bash
npm install -g netlify-cli
netlify deploy --prod --dir=dist
```
